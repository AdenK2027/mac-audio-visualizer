# Mac Audio Visualizer
A real-time music visualizer built with p5.js.

## Setup for Mac
1. Install **BlackHole 2ch**.
2. Create a **Multi-Output Device** in Audio MIDI Setup (Speaker + BlackHole).
3. Set System Input to **BlackHole 2ch**.

## How to Run
1. Open Terminal in this folder.
2. Run `python3 -m http.server`.
3. Open `http://localhost:8000`.
